Determinar cuáles de las siguientes afirmaciones son verdaderas cualesquiera sean los subconjuntos $A$, $B$ y $C$ de un conjunto referencial $V$ y cuáles no. Para las que sean verdaderas, dar una demostración, para las otras dar un contraejemplo.

* i) $(A \triangle B) - C = (A - C) \triangle (B - C)$
* ii) $(A \cap B) \triangle C = (A \triangle C) \cap (B \triangle C)$
* iii) $C \subseteq A \Rightarrow B \cap C \subseteq (A \triangle B)^c$
* iv) $A \triangle B = \emptyset \iff A = B$
